# Поддержка 50 Ом и 75 Ом (PORT-Z)

NanoVNA-X поддерживает работу с системами различного импеданса (например, 75 Ом для ТВ и видео оборудования) при использовании стандартной 50-омной калибровки.

## Как это работает

Прибор аппаратно рассчитан на 50 Ом. Все измерения производятся в 50-омном тракте.
Для работы с 75-омной нагрузкой используется математическая функция **ренормализации импеданса**.

1. Вы проводите калибровку (SOLT) с использованием стандартного 50-омного набора мер (Open, Short, Load 50 Ohm).
    * **Важно:** Всегда выполняйте калибровку с использованием **50-омного набора мер**.
    * Не используйте 75-омную нагрузку для калибровки. Прошивка рассчитывает калибровочные коэффициенты исходя из 50-омного стандарта.
    * Функция ренормализации (`PORT-Z`) математически пересчитывает данные 50-омной калибровки в 75 Ом.

2. В меню прибора меняете импеданс порта (`PORT-Z`) на 75 Ом.
    * Прибор пересчитывает измеренные S-параметры (Gamma) так, как будто измерения проводились прибором с портами 75 Ом.
    * Центр диаграммы Смита смещается в точку 75 Ом.
    * КСВ (SWR) пересчитывается относительно 75 Ом.

## Инструкция: Настройка 75 Ом

1. Выполните стандартную калибровку с 50 Ом нагрузкой.
2. Перейдите в меню: `MEASURE`.
3. Найдите пункт `PORT-Z`. По умолчанию там стоит `50 Ohm`.
4. Нажмите на `PORT-Z` и введите `75` -> `x1`.
5. Теперь все графики (Smith, LogMag, SWR) показывают значения для 75-омной системы.

## Физические ограничения

* Использование 50-омных разъемов и кабелей вносит небольшую погрешность рассогласования, которую математика ренормализации старается компенсировать, но физические отражения на переходах 50/75 остаются.
* Для точных лабораторных измерений рекомендуется использовать согласующие переходы (Matching Pads) 50-75 Ом (Minimum Loss Pad), однако они вносят затухание (около 5.7 дБ), которое нужно компенсировать в калибровке.
* Программный метод `PORT-Z` идеален для быстрой оценки антенн и кабелей 75 Ом без дополнительного "железа".
